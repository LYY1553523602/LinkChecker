# 链接点赞检测器 - 详细使用指南

## 快速开始（5分钟上手）

### 第一步：准备工作

1. **确保手机已安装**：
   - 抖音APP（已登录）
   - 小红书APP（已登录）
   - 今日头条APP（已登录）

2. **安装本应用**

### 第二步：开启无障碍服务

这是最关键的一步！

1. 打开「链接点赞检测器」应用
2. 点击 **「开启无障碍服务」** 按钮
3. 系统会跳转到设置页面
4. 找到 **「链接点赞检测服务」**（通常在「已下载服务」或「更多服务」中）
5. **开启开关**，并在弹窗中点击「确定」
6. 返回应用，按钮应显示「无障碍服务已开启」

> ⚠️ **重要**: 如果不开启此服务，应用无法自动读取点赞数

### 第三步：复制微信聊天记录

1. 打开微信，进入「文件传输助手」
2. 找到包含链接的**合并聊天记录**（长条状的消息）
3. **长按**这条消息
4. 在弹出菜单中选择 **「复制」**

> 💡 **提示**: 如果复制的是多条聊天记录，确保包含链接的文本都在里面

### 第四步：提取链接

1. 切换回「链接点赞检测器」应用
2. 在顶部输入框**长按**，选择「粘贴」
3. 点击 **「提取链接」** 按钮
4. 下方列表会显示所有找到的链接

### 第五步：开始检测

1. 确认链接列表正确
2. 点击 **「开始检测」** 按钮
3. 应用会自动：
   - 打开抖音/小红书/今日头条
   - 读取点赞数
   - 自动返回
   - 继续下一个...

4. **过程中请勿操作手机**，让应用自动运行

### 第六步：查看结果

检测完成后：
1. 会显示统计页面
2. 列表中标绿色的链接是点赞≥50的
3. 点击结果可以查看详情

## 常见问题排查

### 问题1：无障碍服务已开启但检测不到数据

**原因**: 各平台APP版本更新，界面元素变化

**解决方法**:
1. 确保APP是最新版本
2. 尝试手动打开一个链接，观察点赞数的位置
3. 如果位置有变化，需要更新适配代码

### 问题2：链接打开后一直等待，最后显示超时

**原因**: 页面加载慢或无障碍服务未找到点赞数

**解决方法**:
1. 检查网络连接
2. 增加等待时间（修改代码中的 `checkTimeout`）
3. 确保APP已登录

### 问题3：某些链接没有被提取

**原因**: 链接格式不匹配

**解决方法**:
在 `LinkExtractor.kt` 中添加新的正则表达式

### 问题4：应用闪退或卡死

**原因**: 可能是内存不足或无障碍服务异常

**解决方法**:
1. 清理后台应用
2. 重启无障碍服务
3. 减少一次检测的链接数量（建议≤20个）

## 高级使用技巧

### 批量处理大量链接

如果一次有50+链接：
1. 分成多批，每批15-20个
2. 分批粘贴检测
3. 最后汇总结果

### 手动验证结果

自动检测可能有误差，建议：
1. 对重要的链接手动打开验证
2. 关注接近50赞的链接（40-60之间）

### 自定义检测阈值

默认阈值是50，如需修改：
1. 打开 `LinkCheckAccessibilityService.kt`
2. 找到 `isAboveThreshold = number >= 50`
3. 修改 `50` 为你需要的数值

## 技术说明

### 工作原理

1. **链接提取**: 用正则表达式从文本中匹配URL
2. **平台识别**: 根据域名判断所属平台
3. **跳转打开**: 使用 Intent + URL Scheme 打开对应APP
4. **数据读取**: 通过 Android AccessibilityService 读取屏幕上的点赞数文本
5. **结果判断**: 解析文本（支持 "1.2w"、"5万" 等格式），判断是否达标

### 安全性

- 应用不需要网络权限即可工作
- 无障碍服务只读取屏幕文本，不点击或修改内容
- 所有数据保存在本地
