# 链接点赞检测器 (LinkChecker)

一个 Android 应用，用于自动检测微信聊天记录中的链接（抖音、小红书、今日头条、微信文章等）的点赞数量，并筛选出点赞数 ≥50 的链接。

## 功能特点

1. **链接自动提取**: 从微信聊天记录中自动识别并提取各平台链接
2. **多平台支持**: 
   - 抖音 (v.douyin.com)
   - 小红书 (xhslink.com, xiaohongshu.com)
   - 今日头条 (toutiao.com)
   - 微信文章 (mp.weixin.qq.com)
3. **自动检测**: 通过无障碍服务自动打开链接并读取点赞数
4. **结果筛选**: 自动标记点赞数 ≥50 的链接

## 使用步骤

### 1. 安装应用

将项目导入 Android Studio 并构建 APK，或直接使用提供的 APK 安装。

### 2. 开启无障碍服务（重要）

首次使用需要开启无障碍权限：
1. 打开应用，点击「开启无障碍服务」按钮
2. 跳转到系统设置后，找到「链接点赞检测服务」
3. 开启服务，并确认授权

### 3. 复制微信聊天记录

1. 在微信中打开「文件传输助手」
2. 找到包含链接的合并聊天记录
3. **长按**聊天记录，选择「复制」

### 4. 粘贴并提取链接

1. 回到本应用
2. 在输入框中粘贴复制的聊天记录
3. 点击「提取链接」按钮
4. 应用会自动识别并列出所有链接

### 5. 开始检测

1. 确认链接列表无误后，点击「开始检测」
2. 应用会自动：
   - 打开抖音/小红书/今日头条 APP
   - 等待页面加载
   - 通过无障碍服务读取点赞数
   - 自动返回并记录结果
   - 继续下一个链接
3. 检测完成后会显示结果页面

## 注意事项

### 无障碍服务适配

由于各平台 APP 的界面经常更新，无障碍服务可能需要适配。当前版本支持：

- **抖音**: 支持大部分版本，点赞数显示在右侧按钮区域
- **小红书**: 支持大部分版本，点赞数在底部工具栏
- **今日头条**: 支持文章和视频点赞
- **微信文章**: 用内置浏览器打开，尝试读取底部点赞

如果某个平台无法读取，可能是因为：
1. APP 版本太新/太旧，界面元素ID变化
2. 页面加载超时（可调整超时时间）
3. 需要更新无障碍服务的节点查找逻辑

### 时间消耗

每个链接检测大约需要 10-20 秒（打开APP + 加载 + 读取数据）。建议一次检测不超过 20 个链接。

### 手机要求

- Android 6.0 (API 24) 及以上
- 已安装对应的 APP（抖音、小红书、今日头条）
- 各 APP 已登录账号

## 常见问题

### Q: 为什么检测不到点赞数？
A: 可能的原因：
1. 无障碍服务未开启
2. APP 版本不兼容，需要更新适配代码
3. 页面加载超时，可以调整 MainActivity 中的 `checkTimeout` 值

### Q: 能否支持更多平台？
A: 可以，在 `LinkExtractor.kt` 中添加新的正则表达式和平台类型，在 `PlatformHandler.kt` 中添加跳转逻辑，在 `LinkCheckAccessibilityService.kt` 中添加点赞数解析逻辑。

### Q: 检测速度能更快吗？
A: 受限于 APP 加载速度和无障碍服务响应，当前速度已是较优方案。可以缩短 `checkTimeout` 和检测间隔，但可能导致读取失败。
